How to use LocalStorage in JavaScript

What is a LocalStorage?

LocalStorage is a Web Storage API which is used to store data inside the browser. LocalStorage is basically a space through which you can store data (which is in the form of key value pair) locally inside the browser, with the help of this you do not need any database to store your data LocalStorage saves your data even after refreshing the window or closing the server.

How to use LocalStorage in JavaScript?

There are certain commands through which you can get,save, remove and clear data from the LocalStorage.

1. LocalStorage.setItem(key, value) -- This command is used to save data inside the localStorage it has two parameters key and value, Key is the parameter which stores the value inside the localStorage.

For example-- We can save key name like this LocalStorage.setItem("name","Tanishq") so with this the name key get stored inside the browser localStorage with a value of Tanishq


2. LocalStorage.getItem(key) -- This command is used to get the data which is stored into the database. User can easily fetch his data using this command,for example- if we have a key name which stores name "Robin" inside localStorage we can get name Robin using localStorage("name")


3. LocalStorage.removeItem(key) -- This command is used to remove an item from the localStorage, it will remove any key/value pair from the localStorage. For example you can remove name robin by typiing localStorage.removeItem("name")


4. LocalStorage.clear() -- This is used to remove all the data from the localStorage.

Let's go through this example in which I created a signup and login functionality using localStorage.

Signup


    const username=document.getElementById("username")
    const email=document.getElementById("email")
    const password=document.getElementById("password")
    const phone=document.getElementById("phone")

    document.getElementById("submit").addEventListener("click",()=>{
        localStorage.setItem("name",username.value)
        localStorage.setItem("email",email.value)
        localStorage.setItem("phone",phone.value)
        localStorage.setItem("password",password.value)

    })

Login


    const username=document.getElementById("username")
    const email=document.getElementById("email")
    const password=document.getElementById("password")
    const phone=document.getElementById("phone")

    document.getElementById("submit").addEventListener("click",()=>{
        if(localStorage.getItem("email")==email.value 
        && localStorage.getItem("password")==password.value){
            alert("user logged in")
        }
        else{
            alert("sign up first")
        }

    })
