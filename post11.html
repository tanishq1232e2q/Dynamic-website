Best place to host nodejs app

Most of the people choose platforms like Heroku, Github, digitalocean etc and they faced errors everytime when they host nodejs apps, so do you want to host nodejs applications in fast and easy way with better hosting facilities free from all type of errors, so here is the article in which I suggests you the best platform to host nodejs applications with best specifiactions.

The Best platform for this is Render

Render is the best platform which I used to host nodejs app in less amount of time by operating few steps.

Steps to start deploying your app on Render

1. First you need to create a repository on github for this you can just create your account on github after sign in go for new-> Create new repository enter the name of your repository and just create it.

2. Now you need to set up your repository for this just click on uploading an existing file and select the folder which contains all the files of your website after uploading of all files click on commit to successfully apply the changes. Great now you successfully created your repository.

3. Now You can see your uploaded files, just click on code and copy the ssh code that will be used further.

4. Now open render.com render.com and create your account and sign in on the platform now you can see the dashboard click on new->web service to create a web service now you have two options-

i. First step is to connect your github account with render and add your repository on the dashboard to connect.

ii. Second step is simply click on the public git repository and paste your previous ssh code and click on connect.

5. Now you are on final step to deploy a web service for your repository just enter the name and start command(which are used to start nodejs like node form.js) and choose free plan and click on create web service.

Congratulations!! Finally now you can see that your app is deploying on render it will take some time for deployment. You can get your website by clicking on the url at the top of the page named as like "websitename.onrender.com"
